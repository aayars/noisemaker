# scrollX

Horizontal scroll.

## Arguments

### `x`
- **Type:** Number.
- **Default:** `0`.
- **Range:** -10–10.
- **Description:** Horizontal factor or coordinate.
### `speed`
- **Type:** Number.
- **Default:** `0`.
- **Range:** -10–10.
- **Description:** Animation speed.

## Examples

### Positional

```dsl
noise().scrollX(0.1).out()
```

### Keyword

```dsl
noise().scrollX(x: 0.1).out()
```
