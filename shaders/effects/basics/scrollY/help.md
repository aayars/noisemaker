# scrollY

Vertical scroll.

## Arguments

### `y`
- **Type:** Number.
- **Default:** `0`.
- **Range:** -10–10.
- **Description:** Vertical factor or coordinate.
### `speed`
- **Type:** Number.
- **Default:** `0`.
- **Range:** -10–10.
- **Description:** Animation speed.

## Examples

### Positional

```dsl
noise().scrollY(0.1).out()
```

### Keyword

```dsl
noise().scrollY(y: 0.1).out()
```
