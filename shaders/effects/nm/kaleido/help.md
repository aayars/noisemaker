# Kaleido

Toggle kaleidoscope effect on or off.

## Parameters

- **Sides**: Number of kaleidoscope wedges.
- **SDF Sides**: Polygon sides for SDF distance metric (0-2 uses Euclidean).
- **Blend Edges**: Fade to original image at edges.
- **Point Frequency**: Number of Voronoi points per axis.
- **Generations**: Voronoi point cloud generations to accumulate.
- **Distribution**: Voronoi point distribution pattern.
- **Point Drift**: Random drift applied to Voronoi points.
- **Include Corners**: Include corner points for grid distributions.
