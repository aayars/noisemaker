# Worms

Toggle the worms effect on or off.

## Parameters

- **Behavior**: Worm behavior pattern.
- **Density**: Number of agents relative to frame size.
- **Stride**: Agent step length control.
- **Kink**: Controls heading influence from the flow field.
- **Stride Deviation**: Variation in stride applied per agent.
- **Quantize**: Snap worm headings to discrete steps.
- **Trail Intensity**: Fade applied to feedback accumulation (percent).
- **Input Intensity**: Strength of the source texture in the final blend (percent).
- **Lifetime**: Seconds before agents respawn; 0 keeps trails permanent.
