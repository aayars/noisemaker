# solid

Flat color.

## Arguments

### `r`
- **Type:** Number.
- **Default:** `0`.
- **Range:** 0–1.
- **Description:** Red channel multiplier or base value.
### `g`
- **Type:** Number.
- **Default:** `0`.
- **Range:** 0–1.
- **Description:** Green channel multiplier or base value.
### `b`
- **Type:** Number.
- **Default:** `0`.
- **Range:** 0–1.
- **Description:** Blue channel multiplier or base value.

## Examples

### Positional

```dsl
solid(#ff0000).out()
```

### Keyword

```dsl
solid(r: #ff0000).out()
```
