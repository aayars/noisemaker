# a

Isolate alpha channel.

## Arguments

### `scale`
- **Type:** Number.
- **Default:** `1`.
- **Range:** -10–10.
- **Description:** Overall scale of the effect.
### `offset`
- **Type:** Number.
- **Default:** `0`.
- **Range:** -10–10.
- **Description:** Offset amount applied to the effect.

## Examples

### Positional

```dsl
noise().a(0.5, 0.1).out()
```

### Keyword

```dsl
noise().a(scale: 0.5, offset: 0.1).out()
```
