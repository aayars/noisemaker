# Background

Solid color and gradient background generator with rotation and opacity control.

## Usage

```dsl
background(type: 10, color1: #ff0000, color2: #0000ff)
```

## Parameters

*   **type**: The type of background (Solid, Horizontal Gradient, Vertical Gradient, Radial Gradient).
*   **rotation**: Rotation of the gradient in degrees (-180 to 180).
*   **opacity**: Opacity of the background (0-100).
*   **color1**: Primary color.
*   **color2**: Secondary color (for gradients).
